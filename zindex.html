La proprietà Z-INDEX

Abbiamo capito che con CSS è possibile utilizzare determinate tecniche per sovrapporre un elemento su di un altro. Quale elemento tuttavia risulterà essere in sovraimpressione rispetto all'altro? Tipicamente l'ultimo elemento in HTML è anche l'ultimo elemento disegnato dal browser, ma l'ordine di apparizione può venire alterato grazie alla proprietà z-index. Questa proprietà accetta un valore tipicamente numerico e controlla i valori circostanti per stabilire le prorità. Z-index necessita di un posizionamento dell'elemento (cioè non funziona se l'elemento ha position: static).
